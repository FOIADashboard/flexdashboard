How to Update the Dashboard (Will Be Subject to Change as we move from Box to Git/Notion in coming months

  1. Download R 4.33 and RStudio.
  2. Download all files from the FOIA Dashboard Box folder to your computer:
    • FOIA Dashboard Files
    • Ensure that these files are saved to your Desktop (not just the Downloads folder).
  3. Locate the file named flexdashboard_example.Rmd (inside the flex_dashboard folder).
  4. Open flexdashboard_example.Rmd in RStudio.
  5. Make necessary edits/updates:
    • Use the "Run Document" tool to verify your edits.
    • Use Ctrl + Z to undo if needed.
  6. Publish Changes to the website:
    • After running the document, click "Publish".
    • Follow the prompt and log into the ShinyApps account in your browser:
      • Email: deportationresaerchclinic@proton.me
      • Password: (Enter the provided password)
    • Copy and paste the confirmation.
    • Click "Publish!"
